# BlogSpot - Full Project

This archive contains three apps:
- backend (Node.js + Express + MongoDB)
- frontend (React user app)
- admin-dashboard (React admin app)

Quick start:
1. Open terminal
2. cd backend
3. npm install
4. node scripts/create_admin.js
5. npm run dev

Then open frontends in separate terminals:
- cd frontend && npm install && npm start
- cd admin-dashboard && npm install && npm start

Frontend expects backend at http://localhost:5000
